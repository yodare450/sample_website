重要なHTMLタグ一覧

このページは、初心者シリーズWeb基礎編「HTML」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページとして作られます。このページの制作を通じて、HTMLのテーブル（table）をtableタグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ブロック要素（block element）

HTMLタグ タグ名 目的
h1–h6 見出し（heading） この中に見出しを置く（レベル： 1–6）
p 段落（パラグラフ： paragraph） include a paragraph of text
table 表（テーブル: table） この中にテーブルを置く
tr 表の行（table row） この中に行データを置く
th 表の見出し（table header） この中に表の見出しを置く
td 表のデータ（table data） この中に表のセルデータを置く
header ヘッダー（header） この中に冒頭要素を置く
div 区画（division） この中に１区画分の要素を置く
ol 順序付きリスト（ordered list） 順に番号を割り当ててリスト項目を並べる
ul 順序なしリスト（unorderd list） 順序なしでリスト項目を並べる
li リスト項目（list item） この中にリストの項目を置く

インライン要素（inline element）

HTMLタグ タグ名 目的 記述例 画面上の表示k効果
em 軽い強調（emphasized） テキストを軽く強調する 技術の成熟（<em>technical sophistication</em>） 技術の成熟（technical sophistication）
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー ハイパーリンクにする <a href="https://learnenough.com">Learn Enough</a> Learn Enough
img 画像（image） 画像を配置する <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code コード（code） 等幅フォント表示 <code>table</code> table
span スパン（span） 装飾用に文章を区切る あれは<span>何年か</span>前 あれは何年か前

ドキュメントタグ

html html要素（html） HTMLドキュメントであることを示す
head head要素（head） HTMLドキュメントに関する情報を書き込む部分
body body要素 Webページの本体部分
title title要素 Webページのタイトル
meta meta要素 文字符号化の形式を指定
